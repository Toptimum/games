Игра "Alien Invasion"

Описание игры:
Вы управляете космическим кораблем с помощью стрелок на клавиатуре. Стреляете пробелом. Необходимо сбить всех
пришельцев, и побить рекорд.

Возможности игры:
- управляете кораблем и стреляете
- летают пришельцы, которые выстреляют при сбивании
- по мере игры увеличивается скорость пришельцев
- отслеживается нажатие клавиш: стрелочки, пробел, p, q
- выводится счет игры: жизни, рекорд, текущий уровень и счет
- рекордный счет хранится в файле
- играет фоновая музыка
- при попадании воспроизводятся звуки.

В работе над проектом использовал: книгу "Изучаем Python" (Э. Мэтиз), материалы в интернете, звуки для игр, ООП,
графический редактор Photoshop, Python 3.7, библиотеку Pygame, редактор кода PyCharm CE.
